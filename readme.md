# Demoqa Home - Test Automation Project

## Описание
Проект для автоматизации тестирования сайта Saucedemo.com с использованием Selenium и Page Object Model.

## Структура проекта
- `pages/` - Классы страниц (Page Object)
- `tests/` - Тесты
- `components/` - Переиспользуемые компоненты

pip install -r requirements.txt
